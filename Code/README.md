**Prerequisites--------------------------------------------------**



Ensure the following are installed and set up before running this project:

&nbsp;   •    Python

&nbsp;   •    Microsoft Word

&nbsp;   •    EndNote

&nbsp;   •    UiPath

&nbsp;   •    An active OpenAI (ChatGPT) API key



**Configuration Required-----------------------------------------**



***\[ OpenAI ]***



This project requires your own OpenAI / ChatGPT API key.

The code will not run without one. Please note that API access may require a paid plan.



***\[ UiPath ]***



The provided UiPath .xaml workflow was tested on my own device only.

Compatibility may vary depending on your system setup, screen resolution, and UiPath version.

You may need to modify selectors or activities to suit your environment.



***\[ Python ]***



In script.py, locate the main() function near the bottom of the file.



Under the comment:



\### CHANGE FILE PATH ACCORDINGLY ###



Update the following variables to match your local file structure:

&nbsp;   •    base\_path

&nbsp;   •    file\_path

&nbsp;   •    workflow\_path
